
	How to use Chaos Mode
	
	1. BACK UP WORLD BEFORE USING. IT WILL DESTROY YOUR BUILDS.
	2. If using the datapack version, go into your world folder and put the zip file in the datapack folder. If using the mod version, install it like any other mod. 
	3. Run the following command to enable Chaos Mode: /function chaos:general/enable. You will need commands on to use this.
	4. Run the command, /function chaos:timer/start to start the chaos. 
	5. Run the command /function chaos:general/pause and /function chaos:general/resume to pause and resume the timer.
	6. To adjust difficulty, use the following command: /function chaos:general/level_three where 'level_three' can be replaced with 'level_one' to 'level_seven' to manually set the difficulty level,
	or use /function chaos:general/increase_chaos_level and /function chaos:general/decrease_chaos_level to increase or decrease the difficulty from its current level.
	7. Use /function chaos:general/disable to turn Chaos Mode off entirely.
